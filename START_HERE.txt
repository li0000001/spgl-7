🚀 快速开始（只需 4 步，5 分钟）
================================================================================

1️⃣  打开项目
    → 在 Android Studio 中: File → Open
    → 选择这个项目的文件夹
    → 点击 Open

2️⃣  等待 Gradle 同步
    → Android Studio 会自动下载所有必要的工具和依赖
    → 不需要手动操作，耐心等待 3-5 分钟

3️⃣  创建或连接设备
    → 如果没有虚拟设备：Tools → Device Manager → Create Device
    → 或连接你的 Android 手机 (需要启用 USB 调试)

4️⃣  运行应用
    → 点击绿色的 Run 按钮
    → 或按 Shift + F10
    → 选择你的设备
    → 应用启动！

🆘 遇到问题?
================================================================================

1. 查看 ANDROID_STUDIO_SETUP.md 的 FAQ 部分
2. 查看 QUICK_START.md 的常见问题部分
3. 尝试: File → Invalidate Caches → Invalidate and Restart

常见问题解决:
- Gradle 同步失败: ./gradlew clean
- SDK 找不到: File → Project Structure
- 应用崩溃: 检查 Logcat 窗口 (View → Tool Windows → Logcat)
